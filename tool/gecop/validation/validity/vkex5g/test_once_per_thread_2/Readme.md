# Validity VKEX-5G

This [test](.) is exercising the condition [5G](../Readme.md) of the validity rule [VKEX](../../vkex/Readme.md).

### Description

In this test, there is a creation expression with a once-per-thread creation procedure `make`
and a creation type which is a reference type but not separate. This this test `VKEX-5G` is
not violated.
