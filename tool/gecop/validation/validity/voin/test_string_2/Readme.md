# Validity VOIN

This [test](.) is exercising the validity rule [VOIN](../Readme.md).

### Description

In this test, the lower bound `"dΩ"` of the of choice interval in the multi-branch
instruction is greater than the upper bound `"a"`. This violates `VOIN`.

### Notes

* ISE Eiffel (as of 25.12.9.8922 and after) does not support inspect expressions
  whose type is a sized variant of `STRING`.
