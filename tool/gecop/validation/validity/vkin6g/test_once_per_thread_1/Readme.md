# Validity VKIN-6G

This [test](.) is exercising the condition [6G](../Readme.md) of the validity rule [VKIN](../../vkin/Readme.md).

### Description

In this test, there is a creation instruction with a once-per-thread creation procedure `make`
and a creation type which is a reference type but not separate. This this test `VKIN-6G` is
not violated.
