# Validity VOMB-9G

This [test](.) is exercising the condition [9G](../Readme.md) of the validity rule [VOMB](../../vomb/Readme.md).

### Description

In this test, the second choice `{AA}.f .. {AA}.g` is made up of Unique attributes
and the first choice `-2 .. -1` does not contain positive integer constants. In this test
`VOMB-9G` is not violated.
