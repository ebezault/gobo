# Validity VOMB-9G

This [test](.) is exercising the condition [9G](../Readme.md) of the validity rule [VOMB](../../vomb/Readme.md).

### Description

In this test, the first choice `{AA}.f` is a Unique attribute and the
second choice `-1` is not a positive integer constant. In this test
`VOMB-9G` is not violated.
