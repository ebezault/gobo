# Validity VOMB-9G

This [test](.) is exercising the condition [9G](../Readme.md) of the validity rule [VOMB](../../vomb/Readme.md).

### Description

In this test, the first choice `{AA}.f` is a Unique attribute declared in
class `AA`, but the second choice `{BB}.g` is a Unique attribute declared
in another class `BB`. This violates `VOMB-9G`.
