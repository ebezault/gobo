# Validity VYCQ-2

This [test](.) is exercising the condition [2](../Readme.md) of the validity rule [VYCQ](../../vycq/Readme.md).

### Description

In this test, class `BB` declares a conversion query of the form `f: {CC [INTEGER]}`
and `BB` conforms to `CC [ANY]` but does not conforma to `CC [INTEGER]`. In this test
`VYCQ-2` is not violated.
