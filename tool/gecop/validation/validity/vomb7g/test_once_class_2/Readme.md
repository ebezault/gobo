# Validity VOMB-7G

This [test](.) is exercising the condition [7G](../Readme.md) of the validity rule [VOMB](../../vomb/Readme.md).

### Description

In this test, the type of the choice `{detachable BB}.f` in the multi-branch instruction
has the same base class as the type of inspect expression. In this test `VOMB-7G` is
not violated.
