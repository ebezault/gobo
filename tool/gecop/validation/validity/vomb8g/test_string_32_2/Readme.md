# Validity VOMB-8G

This [test](.) is exercising the condition [8G](../Readme.md) of the validity rule [VOMB](../../vomb/Readme.md).

### Description

In this test, the type of inspect expression is separate and controlled.
In this `VOMB-8G` is not violated.

### Notes

* ISE Eiffel (as of 25.12.9.8922 and after) does not support inspect expressions
  whose type is a sized variant of `STRING`.
