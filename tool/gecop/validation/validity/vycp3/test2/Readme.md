# Validity VYCP-3

This [test](.) is exercising the condition [3](../Readme.md) of the validity rule [VYCP](../../vycp/Readme.md).

### Description

In this test, class `BB` declares a conversion procedure of the form `f ({CC [INTEGER]})`
but `CC [INTEGER]` conforms to `BB [ANY]`. This violates `VYCP-3`.
